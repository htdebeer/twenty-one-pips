TopPlayerHTMLElement

TopPlayerHTMLElement

A Player in a dice game. A player's name should be unique in the game. Two different TopPlayerHTMLElement elements with the same name attribute are treated as the same player. In general it is recommended that no two players do have the same color, although it is not unconceivable that certain dice games have players work in teams where it would make sense for two or more different players to have the same color. The name and color attributes are required. The score and has-turn attributes are not.
